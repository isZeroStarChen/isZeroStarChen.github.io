安装虚拟机之Ubuntu

发表于 2021-03-31 更新于 2023-09-17 分类于 Linux

操作系统实验所需的虚拟机可以理解为在一个操作系统中运行另一个操作系统，我们计算机目前普遍使用的都是win10系统(极少数win7)，现在把Ubuntu系统承载在VMware Workstation上来运行。

虚拟机安装之Ubuntu

下载和安装

首先，进行Ubuntu系统安装之前，先下好以下俩个必需品。1>Ubuntu的系统镜像文件<2>VMware Workstation。需要了解的是，VMware Workstation可以使已有操作系统的电脑运行另一操作系统，它起了承载操作系统的作用。而新的操作系统，我们则需要在对应的官网下载对应的光盘镜像文件，然后将其承载在VMware Workstation上(提取码: yrrb)，这里附上了VMware Workstation 16.0 pro和这个版本相匹配的许可证密钥下载。

以下附上步骤操作的截图：第一张是安装好VMware后的截图，然后我们点击“创建新的虚拟机”得到图2，安装Ubuntu的话，我们选择典型操作，然后在安装程序光盘映像文件中把路径选择到你从Ubuntu官网上下载.iso光盘镜像文件，然后选择下一步得到图五，在这里要记得加到系统变量Path里面，安装位置默认是C盘，尽量改到非系统盘，然后下一步一直安装得到了图七和图八(图八是安装之后的VMware所展示的。)

运行配置

在我们开启此虚拟机之后，我们需要对虚拟机进行如下操作：等待Ubuntu自行安装各类包，下载简体中文语言并将其改成预设语言，换源(这只是基础操作，如果想进行一些其他的操作，先下载其所需要的包再进行)

Ubuntu自行安装各类包

这里不做过多阐述，在安装好镜像开启虚拟机之后，就会自动进行安装，稍微等待即可；需要注意的是安装过程中尽量别断网或强制关机，否则可能未安装的程序系统会默认你已经安装，造成不必要的麻烦。
