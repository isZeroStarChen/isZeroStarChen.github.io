第一部分: 填空题

五道填空题做出了三道，另外俩道考图，爷不会，直接白给。。。还是得努力学习。

试题 A: 卡片

看完题目，理解到的题目要求是从数字1一直排过去，看最后那个数字出现的次数先达到2021次。那这样的话，就可以使用Python字符串里读取字符的出现次数的方法，把从1开始排的数字序列放在一个字符串，就如“1234567891011·······”这样的话就可以直接使用字符串的count函数对每个数字进行读取。

简而言之，就是把出现的数字统一存入一个字符串，从0-9的10个数字看作字符，那么题目所给的2021张卡片的限制，其实就是每个字符在字符串中出现的次数不超过2021。

试题A得到的答案是：3181。

试题B 直线

这道题目刚拿到的时候有点懵，大致的意思就是给定一个区域，取区域里的整数坐标，每俩个整数坐标可以确定一条直线，然后题目要你说出这个区域中的整数点可以组成多少条不同的直线。

直线的通常表达式我们可以用y = kx+b表示，k为斜率，b是截距。只要俩条直线的k和b都不同，那就说明俩条直线是不相同的。那么我们就可以通过算斜率和截距来进行对比。但是这种方法有一个缺点就是有俩种特殊情况考虑不到，一种是k = 0，一种是 k 不存在，也就是一条直线平行于y轴、一条平行于x轴。

经过画图之后可以发现，题目所说的2X3的整点，在坐标纸上其实是0 ≤ x < 2, 0 ≤ y < 3这一段范围，而在这段范围中，平行于x轴直线的数目是3，平行于y轴的直线数目是2。也就是说，题目所给的x乘y的整点，平行于y轴的直线数目其实就是x，平行于x轴直线的数目其实就是y。那么我们在俩坐标点不等的情况下，算出俩坐标点的斜率和截距，通过比较这个，来确定直线数目，再加上x和y，就是最后的答案。

那么又有一个问题了，我们算俩坐标点之间直线的截距和斜率，最后通过什么来进行筛选呢，最后还是选择了列表，我们可以把斜率和截距用一个特殊符号组合在一起，再传入列表中(比如“1-2”，斜率为1，截距为2)。列表中可以直接调用set()函数进行去重，最后用列表元素的个数再加上x+y，就是最后的答案(k,b最好保留三位小数，这样方便进行数据处理)。

试题B得到的答案是：40257。

试题C：货物摆放

这道题目理解起来挺简单的，就是把一个数字分成三个因数，然后看这三个因数有多少种情况，那我们可以先循环俩次，把数字先分解成俩个因数，再把俩个因数之中的一个继续分解。但是最后要算的数字太大了，16位数，暴力破解四个小时都不一定跑得出来。

那我们可以考虑采用对所给数字开根号的方式在相对原本数字一半的范围内进行数字的因数计算。在这之前我们需要知道的是，不论分解的数字分成多少个因数，那些因数始终是数字分成俩个因数的列表元素。就比如数字16，它的因数列表是[1,2,4,8,16]，如果把它拆分成三个因数，我们会发现，三个因数都是因数列表中的元素。

第二部分: 编程题

Python编程题的特点感觉就是内存给的贼大，但是时间给的太少。

试题 F: 时间显示

如题，题目大致的意思就是把所给的整数(以毫秒为单位)转化成时分秒。但是在这里要注意的一点则是时不能超过23，也就是小于等于23，如果超过了23亦或者是说24，那么它的时就是00。

那么我们就可以先对所给的整数化成以秒为单位的整数，通过对样例的观察我们发现，可以直接将单位化为秒后小数点之后的数字全部省略，得到以秒为单位的整数X。那么就直接用X整除3600得到整数时H，再用X-3600乘H的值去整除60得到整数分M，最后用X-3600乘H-60乘M得到整数秒S，得到时H、分M、秒S之后，要用时H对24进行取模，以防H大于24，破坏24小时制。最后再以字符串来连接，整合成样例输出的形式。具体代码如下。
